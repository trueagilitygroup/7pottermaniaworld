## 📖 遊戲介紹 About

**7PotterManiaWorld** 是一款以魔法世界為主題的網頁冒險遊戲。玩家將選擇學院、學習咒語、探索城堡，在充滿魔法的世界中展開屬於自己的冒險旅程。

本遊戲為單一 HTML 檔案，無需安裝任何東西，打開瀏覽器即可遊玩。

**7PotterManiaWorld** is a browser-based adventure game set in a magical wizarding world. Players choose their house, learn spells, explore the castle, and embark on their own magical journey.

The entire game is a single HTML file — zero dependencies, zero installation. Just open and play.

## ✨ 功能特色 Features

### 🎮 遊戲系統 Gameplay

| 功能 Feature | 說明 Description |
|:---:|:---|
| 🏠 **四大學院** | 葛萊分多、史萊哲林、雷文克勞、赫夫帕夫，各有獨特屬性加成 |
| 🎭 **NPC 對話系統** | 與 12+ 個角色互動，包含鄧不利多、石內卜、海格、妙麗、馬份、露娜、多比等 |
| ❤️ **人際關係** | 對話選擇影響 NPC 好感度，解鎖隱藏劇情和獎勵 |
| 📜 **多結局主線** | 三種結局：黎明之光（光明）、灰色黎明（中立）、無畏之心（勇氣）|
| 📖 **章節系統** | 四個劇情章節推進，沉浸式劇情體驗 |
| 📍 **15 個場景** | 大禮堂、地窖、圖書館、中庭、天文塔、萬應室、秘密通道、海格小屋、渾拚柳等 |
| 🪄 **9 種咒語** | 路摸思、去去武器走、破心護、咄咄失、速速前、乘乘護法、吼吼燒、水水噴、爆爆炸 |
| ⚔️ **決鬥系統** | 攻擊 / 防禦 / 眩暈三種策略，基於屬性的成敗判定 |
| 📊 **屬性檢定** | 力量、智力、魅力、防禦四大屬性影響劇情走向與對話選項 |
| 📈 **升級系統** | 累積經驗值升級，提升血量與魔力上限 |
| 🎒 **背包系統** | 收集藥水、魔法物品、寶物、線索道具 |
| 📜 **任務追蹤** | 主線+支線任務，進度動態更新 |

### 🎨 技術特色 Technical

| 技術 Tech | 說明 Description |
|:---:|:---|
| 📄 **單檔案** | 所有 HTML / CSS / JS 打包在一個 `.html` 檔案中 |
| 🎵 **程序化音樂** | 使用 Web Audio API 即時生成魔法風格背景音樂，無外部音檔 |
| 🖼️ **SVG 場景** | 所有場景背景畫面皆由 SVG 程式碼繪製 |
| ✨ **Canvas 星空** | 動態閃爍的星空背景動畫 |
| 📱 **響應式設計** | 支援桌面、平板、手機，側邊面板自動轉為抽屜式選單 |
| 🔤 **雙語版本** | 提供繁體中文與英文版本 |
| 🚫 **零依賴** | 不需要 Node.js、npm、或任何框架 |

## 🚀 快速開始 Quick Start

### 方法一：直接開啟

1. 下載 `index.html`
2. 用瀏覽器打開
3. 開始冒險！

### 方法二：GitHub Pages 部署

1. Fork 或 clone 此 repo
2. 進入 repo → **Settings** → **Pages**
3. Source 選擇 **Deploy from a branch**
4. 選擇 **main** 分支，資料夾 **/ (root)**
5. 點擊 **Save**
6. 等待 1-2 分鐘，你的遊戲將上線於：

```
https://你的帳號.github.io/你的repo名稱/
```

## 🎮 遊戲攻略 Game Guide

### 學院選擇建議

| 學院 | 優勢屬性 | 遊玩風格 |
|:---:|:---:|:---|
| 🦁 葛萊分多 | 力量 9 | 擅長正面戰鬥，決鬥勝率高 |
| 🐍 史萊哲林 | 魅力 9 | 社交能力強，容易獲得幫助 |
| 🦅 雷文克勞 | 智力 10 | 解謎與魔藥調製成功率最高 |
| 🦡 赫夫帕夫 | 魅力 10 | 全方位均衡，特殊互動機會多 |

### 💪 屬性提升攻略

每個屬性都有對應的劇情選項或場景動作可以提升。以下是完整的取得方式：

#### ⚔️ 力量（Strength）
力量影響決鬥成功率與物理對抗結果。

| 來源 | 場景 | 條件 / 動作 | 提升 |
|:---|:---:|:---|:---:|
| 決鬥場完美防禦 | 中庭 → 決鬥場 | 防禦 ≥ 6 時施放破心護 | **+1** |

> 💡 建議：選擇葛萊分多獲得起始 9 力量，或在決鬥場練習防禦反擊。

#### 🧠 智力（Intelligence）
智力影響魔藥調製、解謎、以及多個劇情選項。

| 來源 | 場景 | 條件 / 動作 | 提升 |
|:---|:---:|:---|:---:|
| 妙麗的指導 | 圖書館 → 對話 | 「教我幾個實用的咒語吧！」 | **+1** |
| 測奸器眼鏡 | 天文塔 → 露娜 | 「你相信那些傳說嗎？」 | **+1** |
| 中庭練習咒語 | 中庭 | 練習施放咒語 | **+1** |
| 研究古老符文 | 秘密通道 | 已學路摸思 → 觀察符文 | **+1** |
| 重複閱讀典籍 | 禁書區 | 已學咄咄失後再翻開銀光典籍 | **+1** |
| 海格的森林情報 | 海格小屋 | 魅力 ≥ 7 → 詢問禁忌森林 | **+1** |
| 研究星象 | 天文塔 | 研究星象尋找線索 | **+1** |

> 💡 建議：智力是最容易堆疊的屬性，多與妙麗對話、探索秘密通道。

#### 💝 魅力（Charisma）
魅力影響社交對話、NPC 好感度、以及隊友支援的成功率。

| 來源 | 場景 | 條件 / 動作 | 提升 |
|:---|:---:|:---|:---:|
| 萬應室的魔鏡 | 萬應室 | 注視意若思鏡 | **+1** |
| 詢問路平身分 | 鐘樓 | 魅力 ≥ 9 → 詢問他的身分 | **+1** |

> 💡 建議：選擇赫夫帕夫起始 10 魅力，搭配萬應室的魔鏡解鎖路平的祕密身分。

#### 🛡️ 防禦（Defence）
防禦影響受傷減免，以及解鎖某些對話選項。

| 來源 | 場景 | 條件 / 動作 | 提升 |
|:---|:---:|:---|:---:|
| 圖書館研習防禦咒 | 圖書館 | 已學破心護後重複研習 | **+1** |
| 石內卜的筆記本 | 地窖 → 對話 | 誠實回答石內卜 + 已調查蛇形標記 | **+2** |
| 萬應室的光榮之手 | 萬應室 | 拿取閃耀的寶物 | **+2** |

> 💡 建議：誠實對待石內卜可獲得 +2 防禦的筆記本，是早期最大幅的防禦提升。

### 📈 升級獎勵

每次升級會自動提升：
- 血量上限 **+10**
- 魔力上限 **+10**
- 血量與魔力**完全恢復**

### 咒語學習路線

```
起始咒語：路摸思 (Lumos) + 去去武器走 (Expelliarmus)
     │
     ├── 圖書館研習 → 破心護 (Protego)
     ├── 閱讀魔法書 → 速速前 (Accio) [需要智力 ≥ 7]
     ├── 禁書區典籍 → 咄咄失 (Stupefy)
     ├── 禁書區搜尋 → 隨機學會未知咒語
     └── 鐘樓路平教授 → 乘乘護法 (Expecto Patronum) [需要魅力 ≥ 8]

可發現的額外咒語：
     ├── 吼吼燒 (Incendio)
     ├── 水水噴 (Aguamenti)
     └── 爆爆炸 (Confringo)
```

| 😏 跩哥·馬份 | 中庭 / 決鬥場 | 決鬥對手 |
| 📖 妙麗·格蘭傑 | 圖書館 | 場景描述 |

### 小提示

- 🕯️ 學會「路摸思」後再探索黑暗區域，可以發現隱藏寶藏
- 🧪 智力 ≥ 8 可以在石內卜的課堂上成功調製「福來福去」
- 🌙 天文塔可以休息回復血量和魔力
- 🦌 學會乘乘護法後，可以在天文塔向禁忌森林發送信號，人馬翡冷翠會回應你
- 🏡 拜訪海格可以獲得挪威脊背龍蛋或玻璃獸
- 🌳 智力 ≥ 8 才能安全進入渾拚柳下的密道，裡面有劫盜地圖
- 🐾 學會「速速前」後追逐玻璃獸，可以獲得金探子
